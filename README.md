# CryptoAuction - Decentralized Crypto Auction Platform

A modern MERN stack application for cryptocurrency auctions with blockchain simulation, real-time bidding, and comprehensive analytics.

## 🚀 Features

- **Dual Role System**: Separate dashboards for sellers and bidders
- **Real-time Auctions**: Live bidding with instant updates
- **Blockchain Simulation**: Transparent transaction recording
- **Live Crypto Prices**: Real-time cryptocurrency price charts
- **Analytics Dashboard**: Comprehensive auction performance insights
- **Responsive Design**: Mobile-first approach with modern UI
- **Secure Authentication**: JWT-based authentication system

## 🛠️ Tech Stack

### Frontend
- **React 18** - Modern React with hooks
- **React Router** - Client-side routing
- **Framer Motion** - Smooth animations
- **Tailwind CSS** - Utility-first CSS framework
- **Recharts** - Data visualization
- **React Hot Toast** - Toast notifications
- **Parcel** - Fast bundler

### Backend
- **Node.js** - Runtime environment
- **Express.js** - Web framework
- **MongoDB** - NoSQL database
- **Mongoose** - MongoDB object modeling
- **JWT** - Authentication
- **bcryptjs** - Password hashing

## 📦 Installation

1. **Clone the repository**
   ```bash
   git clone https://github.com/yourusername/crypto_bid.git
   cd crypto_bid
   ```

2. **Install dependencies**
   ```bash
   npm install
   ```

3. **Set up environment variables**
   Create a `.env` file in the root directory:
   ```env
   MONGODB_URI=your_mongodb_connection_string
   JWT_SECRET=your_jwt_secret_key
   PORT=5000
   ```

4. **Start the development server**
   ```bash
   npm run dev
   ```

5. **Open your browser**
   Navigate to `http://localhost:1234`

## 🚀 Deployment
